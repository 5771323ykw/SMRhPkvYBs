# 图书管理系统简介 java293

## 项目概述

图书管理系统是一个基于鸿蒙操作系统（HarmonyOS）的前后端项目，旨在实现图书的增删改查等基本管理功能，同时包含了用户登陆与注册模块。

## 技术栈

- **前端语言**：ArkTs
- **后端框架**：SpringBoot
- **数据存储**：MySQL
- **版本**：Api9

## 系统角色

- 系统管理员：负责图书的管理操作
- 普通用户：进行图书查询、借阅等操作

## 运行环境

- 前端：鸿蒙操作系统
- 后端：Java环境，支持SpringBoot的服务器

## 部署步骤

1. 准备Java运行环境及MySQL数据库
2. 部署后端SpringBoot服务
3. 配置前端鸿蒙应用

## 目录结构

```
|-- frontend                # 前端代码目录
|   |-- common              # 通用组件目录
|   |-- pages               # 页面目录
|   `-- app.ts              # 前端入口文件
|-- backend                 # 后端代码目录
|   |-- controller          # 控制器目录
|   |-- service             # 服务层目录
|   |-- repository          # 数据访问层目录
|   `-- Application.java    # 后端启动类
|-- database                # 数据库相关文件目录
`-- README.md               # 项目说明文档
```

## 功能模块

- **用户模块**：包含用户的登录、注册等功能
  - 用户登录
  - 用户注册
- **图书管理**：
  - 图书查询
  - 图书添加
  - 图书修改
  - 图书删除

## 核心亮点

- 基于SpringBoot的后端架构，确保系统稳定性和可扩展性
- 采用MySQL进行数据存储，保障数据的完整性和一致性
